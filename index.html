      이미지 주소 인식

      캡처 이미지를 바로 붙여넣기 캡처 후 여기서 Ctrl+V 또는 ⌘V
      캡처해서 붙여넣는 방법
      Windows

      Win + Shift + S로 캡처 영역을 선택하세요.

      또는 시작 메뉴에서 캡처 도구를 실행해 복사한 뒤 Ctrl + V로 붙여넣습니다.

      Mac

      Control + Command + Shift + 4로 영역을 선택하면 클립보드에 복사됩니다.

      또는 Command + Shift + 5로 스크린샷 앱을 열고 복사한 뒤 Command + V로 붙여넣습니다.

      이미지를 넣으면 주소를 인식합니다
      주소 후보가 없습니다

        도움말

        1. 사무실 주소 설정

        상단 설정에서 사무실 주소를 검색해 선택하세요. 출발·도착지를 사무실로 고정하면 왕복 운행 리포트를 빠르게 만들 수 있습니다.

        2. 경로 입력

        방문지를 직접 입력하거나 이미지로 주소 입력을 눌러 캡처 이미지를 붙여넣으세요. Windows는 Win+Shift+S 또는 캡처 도구를 사용한 뒤 Ctrl+V, Mac은 Control+Command+Shift+4 또는 Command+Shift+5를 사용한 뒤 ⌘V로 붙여넣으면 됩니다.

        3. 순서 조정 및 계산

        경유지 오른쪽 핸들을 드래그해 방문 순서를 바꾸고 경로 계산을 누르세요. 계산 결과는 우측 패널에 구간별 거리와 소요시간으로 표시됩니다.

        팁: 도로명주소, 지번주소, 장소명 모두 입력할 수 있고 OCR 후보는 적용 전에 직접 수정할 수 있습니다.
